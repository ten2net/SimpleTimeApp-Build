# 动态时钟 (Dynamic Clock)

一个显示当前时间的 Android 应用，时间每秒自动更新。

## 功能
- ⏰ 动态时间显示（每秒自动更新）
- 🎨 精美时钟图标
- ✅ 确认按钮
- 🇨🇳 中文应用名称

## 版本历史

### v2.0.0 (最新)
- 时间显示从静态改为动态（每秒更新）
- 添加时钟图标
- 应用名称改为中文"动态时钟"

### v1.0.0
- 初始版本
- 显示当前时间
- 确认按钮功能

## 技术栈
- Kotlin
- Android SDK 34
- Gradle
- Handler/Runnable 定时器

## 本地开发
```bash
./gradlew assembleDebug
```

APK 输出路径: `app/build/outputs/apk/debug/app-debug.apk`

## 安装
1. 下载 APK 文件
2. 在手机上允许"安装未知来源应用"
3. 安装并打开应用
