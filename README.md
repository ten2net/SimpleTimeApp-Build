# Simple Time App

一个显示当前时间的简单 Android 应用。

## 功能
- 📅 显示当前时间（格式：yyyy-MM-dd HH:mm:ss）
- ✅ 确认按钮，点击后更新时间和按钮状态

## 下载 APK
每次提交代码后，GitHub Actions 会自动构建 APK，可在 [Actions](https://github.com/你的用户名/SimpleTimeApp/actions) 页面下载。

## 技术栈
- Kotlin
- Android SDK 34
- Gradle

## 本地开发
```bash
./gradlew assembleDebug
```

APK 输出路径: `app/build/outputs/apk/debug/app-debug.apk`
